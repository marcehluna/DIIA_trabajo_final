# Tareas pendientes

Esta carpeta permite registrar cada tarea pendiente como **un archivo Markdown** independiente (en vez de mantener todo en una única lista).

> Nota: cuando una tarea forma parte de una épica, se guarda dentro de `backlog/epics/<EPIC-ID> - <titulo>/` para mantener jerarquía por carpetas.

## Cómo usarla

- Crear un archivo por tarea con el formato: `BL-YYYY-NNN - titulo-corto.md`
- Completar la plantilla (ver `template.md` en este directorio).
- (Opcional) cuando exista, agregar el link a la Issue de GitHub.

## Nota

Si preferís una vista rápida, seguí usando `backlog/backlog.md` como índice.
